# FLM_numerical

## Overview

This repository implements a **Simultaneous Confidence bands for Functional Linear Model (FLM)** under temporal dependent settings.

---

## Repository Structure

```
FLM_numerical/
├── app/          # Real-data application: electricity demand forecasting
├── simu/         # Simulation studies: benchmarking and method evaluation
└── README.md
```

### `app/` — Real Data Application

Contains R scripts that apply the FLM to real daily electricity demand data with potential daily temperature data as covariates.

### `simu/` — Simulation Studies

Contains R scripts for controlled numerical experiments to evaluate the performance of our confidence bands construction.
